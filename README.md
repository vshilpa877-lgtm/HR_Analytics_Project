# HR_Analytics_Project
HR Analytics Dashboard using Excel, SQL, Power BI, and Tableau

Project Overview:
This project analyzes employee data to identify trends in attrition, employee demographics, job satisfaction, and workforce performance.

Tools Used-
- Excel
- SQL
- Power BI
- Tableau

Key Insights-
- Attrition analysis by department and job role.
- Employee distribution by age and gender.
- Job satisfaction analysis.
- Salary and experience analysis.

Files Included-
- HR_Analytics_Project.xlsx - Source dataset
- HR_ANALYTICS_SQL.sql - SQL queries used for analysis
- Pwer bi.pbix - Power BI dashboard
- Tableau Project.3.twbx - Tableau dashboard

Skills Demonstrated-
- Data Cleaning
- SQL Queries
- Data Visualization
- Dashboard Development
- Business Insights
